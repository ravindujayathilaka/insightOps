# InsightOps

Internal analytics platform for private dashboards and admin tools.

## Features
- Internal page tracking
- Feature usage analytics
- API-key based tracking
- Lightweight JS snippet

## Tech Stack
- Next.js
- Node.js + Express
- PostgreSQL
- Prisma
- Docker

## Setup
1. Copy `.env.example` to `.env`
2. Run `docker compose up -d`
3. Run API and frontend
